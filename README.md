# CubixDLC - Granny Legacy

DLC для Granny Legacy с модфункциями и красивым HUD.

## Описание

Это x64 C++ DLL для инжекта в Unity DX11 приложение (Granny Legacy). DLL предоставляет функционал модов с современным интерфейсом в стиле Exosware ClickGUI.

## Особенности

- ✅ Прямой инжект в Unity DX11 приложение
- ✅ Хуки DirectX 11 через COM vtable (стабильно работает с ASLR)
- ✅ Красивый ClickGUI интерфейс с перетаскиваемыми панелями
- ✅ Водяной знак (watermark) с отображением FPS
- ✅ Звуковые эффекты при включении/выключении модулей
- ✅ Поддержка изменения разрешения и Alt+Enter

## Структура проекта

```
├── assets/
│   └── sounds/          # Звуковые файлы для модулей
│       ├── func_on.wav  # Звук включения модуля
│       └── func_off.wav # Звук выключения модуля
├── hook_render/         # Модуль хуков DirectX 11
├── hud/                 # ClickGUI интерфейс
├── watermark/           # Водяной знак с FPS
├── modules/             # Модули модов (заготовка)
├── deps/                # Зависимости (ImGui, MinHook)
├── dllmain.cpp          # Точка входа DLL
├── CMakeLists.txt       # Файл конфигурации CMake
└── build.bat            # Скрипт для сборки

```

## Требования

- Windows 10/11 (x64)
- CMake 3.16 или выше
- Visual Studio 2019/2022 с поддержкой C++17
- DirectX 11

## Сборка

1. Убедитесь, что у вас установлен CMake
2. Запустите `build.bat` для автоматической сборки
3. Скомпилированная DLL будет находиться в `build\bin\Release\cubixdlc.dll`

### Ручная сборка

```bash
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

## Использование

1. Скомпилируйте проект (см. раздел "Сборка")
2. Скопируйте `cubixdlc.dll` в папку с игрой
3. Скопируйте папку `assets` в ту же директорию, где находится DLL
4. Используйте инжектор для загрузки DLL в процесс игры
5. Нажмите **Del** для открытия/закрытия меню

## Интерфейс

### ClickGUI

- **Перетаскивание панелей**: Зажмите левую кнопку мыши на заголовке панели и перетащите
- **Включение модулей**: ЛКМ на модуле
- **Раскрытие настроек**: ПКМ на модуле (если есть настройки)
- **Прокрутка**: Колесо мыши в области модулей

## Технические детали

### Хуки

- `IDXGISwapChain::Present` (vtable index 8) - основной хук для рендеринга
- `IDXGISwapChain::ResizeBuffers` (vtable index 13) - хук для обработки изменения разрешения

### Используемые библиотеки

- **ImGui** - GUI библиотека для создания интерфейса
- **MinHook** - библиотека для создания хуков через vtable
- **DirectX 11** - для рендеринга
